# Plan 03-01: Memory Graph Fusion

## Objective
Implement global memory graph fusion across STM, LTM, KG, and MM layers with unified query interface.

## What This Builds
A `MemoryFusionService` that overlays all memory layers and provides cross-layer query results merged by relevance score.

## Files Modified
- `backend/src/services/memory_fusion.rs` (new)
- `backend/src/services/mod.rs` (updated)
- `backend/src/routers/memory.rs` (new endpoint)

## Tasks

### Task 1: Create MemoryFusionService
**read_first:** `backend/src/services/memory_orchestrator.rs`, `backend/src/db/stm.rs`, `backend/src/db/ltm.rs`

**action:**
Create `backend/src/services/memory_fusion.rs` with:
- `struct MemoryFusionService { stm: STMRepository, ltm: LTMRepository, kg: KGRepository, mm: MMRepository }`
- `impl MemoryFusionService { fn query(&self, query: &str, tenant_id: &str) -> FusionResult }`
- `enum FusionResult { LayerResults { stm: Vec<MemoryEntry>, ltm: Vec<MemoryEntry>, kg: Vec<MemoryEntry>, mm: Vec<MemoryEntry> }, MergedResults(Vec<MergedEntry>) }`
- Cross-layer query fans out to all repositories and merges by relevance score
- Use semantic similarity for LTM/KG, exact match for STM

**acceptance_criteria:**
- `backend/src/services/memory_fusion.rs` exists with `MemoryFusionService` struct
- `query()` method returns `FusionResult` with both layer-separated and merged results
- Service is added to `services/mod.rs` exports

### Task 2: Add Fusion Status Endpoint
**read_first:** `backend/src/routers/memory.rs`, `backend/src/services/memory_fusion.rs`

**action:**
Add to `backend/src/routers/memory.rs`:
```rust
async fn fusion_status(
    Extension(ctx): RequestTenantContext,
) -> Json<FusionStatusResponse> {
    let fusion = MemoryFusionService::new(/* repos */);
    Json(fusion.get_status())
}
```
Expose `GET /api/v1/memory/fusion/status` returning layer health and entry counts.

**acceptance_criteria:**
- `GET /api/v1/memory/fusion/status` returns JSON with `stm_count`, `ltm_count`, `kg_count`, `mm_count`
- Router registered in `axum_routers/mod.rs`
