# Plan 03-03: Self-Healing Runtime

## Objective
Implement autonomous diagnosis and constrained self-healing for fault recovery.

## Tasks

### Task 1: Create SelfHealingService
**action:**
Create `backend/src/services/self_healing.rs`:
- `SelfHealingService` with `check_health()` returning `HealthStatus`
- Recovery strategies: restart_layer, clear_stale, reload_backup
- Max 3 attempts per fault, exponential backoff

**acceptance_criteria:**
- Service has `check_health()` returning layered health status

### Task 2: Health check endpoint
**action:**
Add `GET /api/v1/memory/health` returning per-layer health.

### Task 3: Tests
**action:**
Create `backend/tests/self_healing.rs` with `test_health_check`, `test_recovery_attempts_limited`

**acceptance_criteria:**
- `cargo test self_healing` passes
