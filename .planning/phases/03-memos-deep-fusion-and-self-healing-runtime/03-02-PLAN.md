# Plan 03-02: Weight Decay & Semantic Evolution

## Objective
Implement observable and tunable weight-decay behavior for semantic evolution.

## Tasks

### Task 1: Add decay config and WeightDecayService
**action:**
Add to `config.toml`:
```toml
[memory_evolution]
decay_lambda = 0.01  # exponential decay rate
```
Create `backend/src/services/weight_decay.rs`:
- `WeightDecayService::new(lambda: f64) -> Self`
- `fn apply_decay(&self, entry: &MemoryEntry, age_seconds: u64) -> f64`
- Formula: w(t) = w0 * e^(-λt)

**acceptance_criteria:**
- `config.toml` contains `[memory_evolution]` section
- `WeightDecayService` applies exponential decay

### Task 2: Add weight status endpoint
**action:**
Add `GET /api/v1/memory/weights/status` returning:
```json
{ "decay_lambda": 0.01, "active_weights": [...] }
```

**acceptance_criteria:**
- Endpoint returns JSON with decay_lambda and active weights

### Task 3: Tests
**action:**
Create `backend/tests/weight_decay.rs` with `test_exponential_decay_formula`

**acceptance_criteria:**
